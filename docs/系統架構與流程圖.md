### 2. 轉接真人策略

Agent 遵循「自主解決優先」原則，資訊不足時的處理順序：

1. 先提供能找到的部分資訊
2. 嘗試用不同關鍵字或角度重新檢索
3. 根據專業知識提供通用建議或常見做法
4. 建議使用者提供更具體的資訊
5. 只有當使用者明確表示以上方式都無法滿足需求時，才詢問是否需要轉接真人客服

`transfer_to_human` 工具僅在以下情況使用：
- 使用者明確堅持要求轉接真人客服
- 涉及安全風險（門鎖無法上鎖、疑似被破壞）
