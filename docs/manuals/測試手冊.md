# RAG 手動測試紀錄

## 測試環境 / 前置條件

1. 確認 pgvector 容器運行中：
   ```bash
   docker ps --filter name=lock_AI --format "table {{.Names}}\t{{.Status}}\t{{.Ports}}"
   ```

2. 確認 vector extension 已啟用：
   ```bash
   docker exec -it lock_AI psql -U lock -d lock_AI_data -c "SELECT extname FROM pg_extension WHERE extname = 'vector';"
   ```

3. 確認資料已 seed（各 collection 有資料）：
   ```bash
   docker exec -it lock_AI psql -U lock -d lock_AI_data -c "SELECT collection_id, COUNT(*) FROM langchain_pg_embedding GROUP BY collection_id;"
   ```

4. 連線資訊：`localhost:5433`，使用者 `lock`，資料庫 `lock_AI_data`
